🚚 Supply Chain Logistics Analysis

📌 Project Overview

This project analyzes supply chain logistics data using Microsoft Excel. It demonstrates data cleaning, Pivot Tables, Pivot Charts, Slicers, and an interactive dashboard to derive business insights.

🎯 Objective

* Analyze supplier performance.
* Evaluate total order value by market segment.
* Track monthly order value trends.
* Measure customer satisfaction across different market segments.
* Build an interactive dashboard for business decision-making.

🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Dashboard Design
* Data Analysis

📊 Dashboard Features

* Total Order Value by Market Segment
* Total Order Value by Supplier
* Average Customer Satisfaction by Market Segment
* Monthly Order Value Trend
* Interactive Slicers for filtering data

📁 Repository Contents

* `Supply_Chain_Logistics_Analysis.xlsx` – Excel workbook containing raw data, Pivot Tables, Pivot Charts, and Dashboard.
* Dashboard screenshots
* README.md

📈 Key Insights

* Compared order values across different market segments.
* Identified top-performing suppliers.
* Analyzed customer satisfaction levels.
* Observed monthly order value trends using interactive visualizations.
